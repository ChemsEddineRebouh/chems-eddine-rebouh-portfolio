## 📝 À propos

Code source de mon portfolio interactif mettant en valeur mes projets et mes compétences en développement. L'interface inclut un mode sombre/clair et est bilingue (Français/Anglais).

## 🛠 Technologies utilisées

* **Frontend** : React 19, TypeScript
* **Stylisation** : Tailwind CSS v4
* **Animations** : Framer Motion
* **Outils de build** : Vite
* **Services** : EmailJS (Formulaire de contact)
* **Déploiement** : Vercel

## ✨ Fonctionnalités

* **Bilingue** : Support complet du français et de l'anglais.
* **Mode Sombre/Clair** : Adaptation au thème du système avec bascule manuelle.
* **Design Responsif** : Interface fluide adaptée à tous les écrans.
* **Formulaire de contact** : Intégration d'EmailJS pour l'envoi direct de courriels.
* **Animations** : Transitions fluides au défilement.

## 🚀 Installation locale

1. Cloner le dépôt :
```bash
git clone [https://github.com/ChemsEddineRebouh/portfolio.git](https://github.com/ChemsEddineRebouh/portfolio.git)
cd portfolio
```

2. Installer les dépendances :
```bash
npm install
```

3. Configurer les variables d'environnement :
Créer un fichier `.env` à la racine du projet et y ajouter les clés EmailJS :
```env
VITE_EMAILJS_SERVICE_ID="votre_service_id"
VITE_EMAILJS_TEMPLATE_ID="votre_template_id"
VITE_EMAILJS_PUBLIC_KEY="votre_public_key"
```

4. Lancer le serveur de développement :
```bash
npm run dev
```

## 🌐 Déploiement

Ce projet est configuré pour être déployé sur **Vercel**. Ajoutez les variables d'environnement `VITE_EMAILJS_*` dans les paramètres de votre projet sur le tableau de bord Vercel avant le déploiement.
